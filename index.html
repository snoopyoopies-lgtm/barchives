FAQ

Questions worth answering.

Do I need a paid Notion plan?

No. Both templates work on Notion's free plan. The chart blocks used in the dashboards are available to all users.

What if I'm on Android?

The Notion templates work anywhere Notion does. The included shortcut is iOS-only (Apple Shortcuts is an Apple-only app), but you can still log entries by adding rows directly in Notion. An Android equivalent using Tasker or HTTP Shortcuts is on the roadmap.

How long does setup actually take?

About 5 minutes for the template + Notion API integration token. The shortcut import is one tap. The full setup guide is included as a Notion page inside the template.

Can I customise the categories, exercises, accounts?

Yes — everything is a Notion property, so adding, renaming, and removing options works exactly as it does in any database. The shortcut reads category and exercise lists from your Notion data, so changes flow through automatically.

What's the difference between "template only" and "template + shortcut"?

Template only includes the full Notion workspace duplicate. Template + shortcut adds the pre-built Apple Shortcut and a video walkthrough of the API integration setup.

Is my financial or training data shared anywhere?

No. The shortcut communicates directly with your own Notion workspace via your personal API token. No servers, no analytics, no third parties involved.

Updates and support?

One-time purchase, lifetime updates. Updates are pushed by re-duplicating the template; your existing data is preserved. Reach out by email for setup help.
